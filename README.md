Download etmek icin:
Sag orta-ust'te cikan "Clone or Download" butonuna basin. Indirilen dosya .zip ise onu acin. Sonuc klasorunu Unity dosyalarinizi tuttugunuz klasore tasiyin.

Dosyayi download ettikten sonra Unity ile acin.
